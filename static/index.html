🌐 Как опубликовать лендинг

    

      

        
Вариант 1: Tilda (бесплатно)

        
    
          
  1. Скачайте HTML файл кнопкой «💾 HTML»
    2. 
          
  2. Войдите на tilda.cc → Создать сайт
    3. 
          
  3. Добавьте блок «HTML/JS/CSS» (в нулевом блоке или Zero Block)
    4. 
          
  4. Вставьте содержимое HTML в поле кода
    5. 
          
  5. Опубликуйте → получите ссылку вида yourdomain.tilda.ws
    6. 
        

      

      

        
Вариант 2: GitHub Pages (бесплатно, ссылка навсегда)

        
    
          
  1. Создайте репозиторий на github.com
    2. 
          
  2. Загрузите HTML файл как index.html
    3. 
          
  3. Settings → Pages → Deploy from branch (main)
    4. 
          
  4. Ссылка: username.github.io/repo-name
    5. 
        

      

      

        
Вариант 3: Netlify Drop (30 сек)

        
    
          
  1. Зайдите на app.netlify.com/drop
    2. 
          
  2. Перетащите HTML файл в окно браузера
    3. 
          
  3. Готово! Ссылка появится сразу (вида random-name.netlify.app)
    4.
